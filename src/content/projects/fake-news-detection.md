## Overview
In the age of information overload, verification is key. This platform helps media houses and readers authenticate news articles by analyzing linguistic patterns and cross-referencing known reliable sources.

## Key Features
- **Url Analysis**: Checks domain reputation and history.
- **Stance Detection**: Determines if a headline matches the body content's claims.
- **Browser Extension**: A plugin that scores articles in real-time as users browse.
- **Crowd-Sourced Labels**: Allows community voting to augment the AI's training data.

## Technical Specifications
- **Deep Learning**: Long Short-Term Memory (LSTM) networks for sequence processing.
- **Database**: MongoDB for flexible storage of article metadata.
- **API**: RESTful API for third-party integrations.
