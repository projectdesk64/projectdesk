## Overview
Creating safe online communities is harder than ever. This Cyberbullying Detection system monitors user-generated content in real-time to identify harassment, hate speech, and toxic comments, allowing moderators to act swiftly.

## Key Features
- **Real-Time Filtering**: APIs that classify text as safe or toxic in milliseconds.
- **Multi-Class Classification**: Distinguishes between hate speech, offensive language, and innocuous profanity.
- **Reporting System**: Users can view flagged content and moderation logs.
- **Tunable Sensitivity**: Admins can adjust strictness levels based on community guidelines.

## Technical Specifications
- **Core Model**: Fine-tuned BERT model for superior context understanding.
- **API**: Lightweight Flask API designed for easy integration into existing social platforms.
- **Visualization**: Vue.js dashboard for moderators to review flagged content.
